	1. pip install cx_Freeze

	2. create a folder and put main python file with icon file and also add application icon folder.

	3. create a setup.py file and paste it where your python file is located (also provide setup file with this file)

	4. then open command prommt where you have created the setup file

	5. command: python setup.py bdist_msi
	
	If you want a setup file to install the software then follow this step

	1. download inno setup software and install it

	2. open inno setup and select the python execlutable file 

	3. then select the folder which all file to execute the python file
